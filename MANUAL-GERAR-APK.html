📱 Manual: Gerar APK do DevMobile

Versão 2.8.0 — Independente do Replit

🎯 O que vai acontecer: Você vai entrar no site da Expo, apertar alguns botões e em ~10 minutos terá o APK pronto para instalar no celular. Não precisa instalar nada no computador.

PASSO 1 — Entrar no site da Expo

1 Abra o navegador e acesse:

🌐 expo.dev

2 Clique em Sign In no canto superior direito

3 Entre com a conta: maikons-individual

PASSO 2 — Abrir o projeto DevMobile

4 Depois de entrar, clique no menu Projects

5 Procure o projeto chamado app-ide e clique nele

Não encontrou? Acesse direto: expo.dev/accounts/maikons-individual/projects/app-ide

PASSO 3 — Iniciar o Build do APK

6 No menu do projeto, clique em Builds

7 Clique no botão New build

8 Vai abrir uma tela com opções. Configure assim:

  • Platform: selecione Android
  • Build profile: selecione preview
  • Deixe o resto como está

9 Clique em Build para confirmar

PASSO 4 — Aguardar o Build

10 A tela vai mostrar o progresso do build

⏱️ Tempo médio: 10 a 15 minutos
Você pode fechar o navegador — o build continua na nuvem da Expo.
Volte quando quiser para ver se terminou.

11 Quando aparecer ✅ Finished, clique no botão Download

12 Salve o arquivo .apk no celular e instale!

PASSO 5 — Instalar o APK no Celular

13 No Android: vá em Configurações → Segurança → Fontes desconhecidas e ative

14 Abra o arquivo .apk que baixou e toque em Instalar

✅ Pronto! O DevMobile está instalado e funciona sem precisar do Replit.

Usando a IA no APK (sem pagar nada)

💡 O APK usa sua própria chave de IA — sem depender do Replit. A Groq e o Gemini são gratuitos!

Para pegar uma chave Groq (grátis):
  1. Acesse console.groq.com
  2. Crie uma conta gratuita
  3. Vá em API Keys → Create API Key
  4. Copie a chave (começa com gsk_...)
  5. No DevMobile, vá em Configurações → IA e cole a chave

Para pegar uma chave Gemini (grátis):
  1. Acesse aistudio.google.com/apikey
  2. Entre com sua conta Google
  3. Clique em Create API Key
  4. Copie a chave (começa com AIza...)
  5. No DevMobile, vá em Configurações → IA e cole a chave
⚠️ Se der algum problema no build:
Se o build falhar, tente de novo — às vezes o servidor da Expo fica ocupado. O keystore (assinatura) já está salvo, não precisa criar de novo.
